TeamBalance verwendet das gleiche Bewertungssystem, dem professionelle Schachspieler und Elite-Sportligen vertrauen. Jetzt für jeden verfügbar.

Keine Diskussionen mehr. Keine Zeitverschwendung mehr. Nur mathematisch ausgewogene Teams.
